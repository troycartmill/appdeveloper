Mobile App Developer, built on the Genesis theme framework. 

This theme is free to download and use for personal use, a client, a website, but not for resale.

<strongAPPDEVELOPER CHILD THEME</strong>
<a href="http://github.com/troycartmill/appdeveloper">http://github.com/troycartmill/appdeveloper</a>

<strong>INSTALL</strong>
1. Upload the AppDeveloper child theme folder via FTP to your wp-content/themes/ directory. (The Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the AppDeveloper theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking.

<strong>WIDGET AREAS</strong>
Primary Sidebar - This is the primary sidebar if you are using the Content/Sidebar or Sidebar/Content Site Layout option.
Home Featured Picture - This is the Home Featured Picture on the homepage.
Home Call to Action - This is the Call to Action at the top of the homepage.
Home Services - This is the services section at the top of the homepage.
Footer #1 - This is the first column of the footer section.
Footer #2 - This is the second column of the footer section.
Footer #3 - This is the third column of the footer section.
Footer #4 - This is the Fourth column of the footer section.
